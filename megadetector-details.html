Megadetector - Wildlife Conservation Framework

A collaborative PyTorch-based framework for wildlife detection and classification in images and videos, designed to support biodiversity studies and conservation efforts.

Key Features

  • Video Processing: Processes video files frame-by-frame with progress tracking and codec customization.
  • Post-Processing: Annotates images, crops detections, and saves results in JSON format.
  • Custom Classifiers: Includes species-specific classifiers such as:
    • Amazon Rainforest Animal Classifier: Recognizes 36 animal species.
    • Opossum Classifier: Specialized in distinguishing opossums from non-opossums.
  • Deep Learning Backbone: ResNet-based architecture with pre-trained and custom-trained weights.
  • Scalable Deployment: Supports batch processing for large datasets.

Technical Details

  • Programming Language: Python
  • Framework: PyTorch
  • Dependencies: numpy, torch, torchvision, tqdm, Pillow, gradio
  • Techniques: Supervised learning, video processing, and visualization tools.

Applications

  • Wildlife monitoring and conservation projects.
  • Automatic species identification for biodiversity studies.
  • Environmental impact assessments.
