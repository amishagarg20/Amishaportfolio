Netflix Userbase Analysis and Visualization Dashboard

An interactive dashboard that analyzes Netflix's userbase demographics, subscription trends, and revenue distribution using Dash and Plotly.

Objectives

  • Analyze and visualize the demographic distribution of Netflix's userbase.
  • Calculate metrics like subscription duration and total revenue per user.
  • Create interactive visualizations to explore user activity, subscription types, and geographic data.

Key Features

  • Data Analysis: Cleaned and preprocessed Netflix user data, including handling dates and missing values.
  • Interactive Dashboard: Built using Dash and Plotly with dropdown filtering for insights.
  • Visualizations: Dashboards, bar charts, and grouped bar charts to analyze user activity and revenue.

Tools and Technologies

  • Python: For data preprocessing and analysis.
  • Dash and Plotly: For interactive dashboard creation.
  • Pandas: For data manipulation and cleaning.
  • Data Sources: Netflix user data file with demographic, subscription, and revenue information.
