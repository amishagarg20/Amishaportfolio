New York Taxi Data Integration

This project focuses on analyzing New York taxi trip data to optimize data integration and build predictive models for trip fare estimation. Leveraging Python and SQL, it handles datasets with 28 million rows, performs exploratory data analysis, and implements ERD in MySQL.

Key Features

  • Analyzed 28M rows of taxi trip data, performing EDA to uncover insights about ride patterns.
  • Utilized SQL and Python for data cleaning, transformation, and integration.
  • Designed and implemented an Entity-Relationship Diagram (ERD) in MySQL for effective database management.

Tools and Technologies

  • Programming: Python
  • Database: MySQL
  • Libraries: Pandas, Matplotlib, Seaborn
  • Visualization: Tableau (optional)

Visualizations

Challenges

  • Handling 28M rows in memory-constrained environments.
  • Cleaning and transforming data with significant variability in ride patterns.

Outcome

This project highlights the ability to manage large-scale datasets, design efficient data pipelines, and deliver actionable insights to improve transportation and logistics systems.
