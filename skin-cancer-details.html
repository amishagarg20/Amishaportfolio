Skin Cancer Classification using CNNs with Soft Attention and Augmented Data

A deep learning solution to classify seven types of skin cancer using the HAM10000 dataset. Leveraged CNNs, soft attention, and data augmentation to achieve an accuracy of 97.75%.

Highlights

  • Dataset: Processed 10,015 dermatoscopic images, resolving class imbalance with oversampling and augmentation.
  • Model: Designed CNN with attention mechanism to focus on critical image regions.
  • Evaluation: Achieved test accuracy of 97.75% on augmented data, leveraging AWS EC2 for training.

Key Responsibilities

  • Data preprocessing and augmentation to enhance class distribution.
  • Implemented a CNN with attention mechanism for improved interpretability.
  • Evaluated models using metrics like F1-score, precision, and recall.

Technical Details

  • Tools: TensorFlow, Keras, AWS EC2 (NVIDIA GPUs), matplotlib, seaborn.
  • Architecture: 4 convolutional layers, max-pooling, 3 dense layers for classification.
  • Techniques: Soft attention mechanism, oversampling, and augmentations (rotations, flips).
